Cara Konversi Suhu


Dari Celsius (°C) ke Fahrenheit (°F)

Suhu dalam derajat Fahrenheit (°F) sama dengan suhu dalam derajat Celsius (°C) dikalikan dengan 9/5 ditambah 32.

S(°F) = ( S(°C) x 9/5 ) + 32
atau
S(°F) = ( S(°C) x 1.8 ) + 32


Dari Fahrenheit (°F) ke Celsius (°C)

Suhu dalam derajat Celsius (°C) sama dengan suhu dalam derajat Fahrenheit (°F) dikurangi 32, kemudian dikalikan dengan 5/9.

S(°C) = ( S(°F) - 32 ) x 5/9
